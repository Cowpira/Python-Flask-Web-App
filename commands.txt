Commands to Configure the Python + Flask Project

1º - pip3 instal virtualenv
    # This command configure the project so we can work in a team and allow other users to 
    have access to the project.

2º - virtualenv env 
    # Create the ENV folder into the project folder

3º - source env/bin/activate
    # command to set the terminal to work in the env project enviroment
    and activate env

4º - pip3 install flask flask-sqlalchemy
    # install flask framework and sql alchemy

5º - from app import db
    # set the interctive shell for flask db terminal
